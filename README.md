# Juanbing (卷饼)

自用的问卷调查与数据收集平台。

## 技术栈

- **框架**: [Next.js](https://nextjs.org/) (App Router)
- **语言**: [TypeScript](https://www.typescriptlang.org/)
- **样式**: [Tailwind CSS](https://tailwindcss.com/)
- **组件**: [Shadcn UI](https://ui.shadcn.com/)
- **数据库**: SQLite

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

应用默认运行在 [http://localhost:3000](http://localhost:3000)。

## 构建

```bash
npm run build
```
